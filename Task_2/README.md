# Laundry Wallah / Laundry Mart - HTML Task

## Project Description

This project is a simple HTML webpage created for the Laundry Wallah / Laundry Mart assignment. It showcases a basic laundry service website using only HTML.

## Features

- Welcome heading
- About section
- Laundry image
- Services section
- Price list displayed in a table
- Book Now form
- Footer with contact information

## Technologies Used

- HTML5

## Folder Structure

```
Task2/
├── index.html
└── README.md
```

## How to Run

1. Download or extract the ZIP file.
2. Open the project folder.
3. Double-click the `index.html` file, or open it in any modern web browser such as Google Chrome, Microsoft Edge, or Mozilla Firefox.
4. The Laundry Wallah / Laundry Mart webpage will be displayed.
