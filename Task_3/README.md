# CSS Selectors Task

## Project Files

- index.html
- style.css

## How to Run

1. Download the ZIP file.
2. Extract (unzip) the ZIP file to any folder on your computer.
3. Open the extracted folder.
4. Make sure `index.html` and `style.css` are in the same folder.
5. Double-click `index.html`, or right-click it and choose **Open with** any web browser (Google Chrome, Microsoft Edge, Firefox, etc.).
6. The webpage will open and display the completed CSS Selectors task.
